*Part A: the 10 platforms the user named (what to steal). Part B: the direct competitive set (where the whitespace and moat are). Grounded in `platforms-comparison.md`, `competitors-direct.md`, `market-legal-ftc.md`.*

## Part A — The 10 platforms (why users choose/stay, growth loops, monetization, what to steal)

Social Perks is **not** a social network — it's a B2B2C tool that rides *on top* of these. So the analysis is "what mechanic do we steal," not "how do we compete."

| Platform | Why users choose/stay | Strongest growth loop | Strongest monetization | What Social Perks steals |
|---|---|---|---|---|
| **TikTok** | #1 organic reach for visual SMBs; 49% use it as a search engine | Algorithmic FYP reach for any creator | Ads + creator fund + shop | **Highest-ROI advocacy platform for visual local business** — but force the Sept-2025 commercial-content toggle on every action |
| **Instagram** | 67% of Gen Z / 54% of 25–34 use it to explore *local* businesses (beats Google Maps for 18–24) | Stories/Reels + tag/mention | Ads + shopping | **The wedge action: Story-tag** (~10s, lowest effort, legal with Paid Partnership label). Verification gated since Meta killed Basic Display API Dec 2024 = moat |
| **YouTube** | Depth, search longevity | Subscriptions + algorithm | AdSense + memberships | Less relevant for local; long-form not the advocacy unit |
| **Patreon** | Creators monetize true fans | Membership + tiers | Take-rate on memberships | **Named status tiers** (Regular→Insider→Ambassador), founding-member framing, **Autopilot churn-saves** (inactive 45d → bounce-back perk) |
| **Discord** | Community + status (roles) | Server invites | Nitro (paid status) | **Visible status/roles** — people pay for status; status substitutes for cash rewards |
| **LinkedIn** | Professional status | Network invites | Recruiter/ads/premium | **Status-as-reward** — proves cash isn't always needed; public credit motivates |
| **Reddit** | Authentic community | Subreddit + search | Ads | **Cautionary:** undisclosed incentivized recommendations are FTC + cultural suicide → **hard-block Reddit actions** (and market the block) |
| **X (Twitter)** | Real-time reach | Retweets | Ads + subs | Share-intent for "win cards" (once OG renders) |
| **Beehiiv** | Newsletter growth | **Boosts** (paid cross-promo) + milestone referrals | Subscriptions + Boosts | **Milestone referrals** ("refer 3 → unlock big perk") + **paid Boosts between neighboring businesses** |
| **Substack** | Writer-reader trust | **Recommendations network** | Subscriptions | **THE one to steal: localized Recommendations** — "supporters of Luna Cafe also earn at Iron Gym." Drove the majority of Substack's new subs; the only compounding local network effect no loyalty competitor has |

**Synthesis — mechanics to steal, ranked:** (1) Substack Recommendations → local cross-business network (the moat); (2) Duolingo streaks + Patreon status → behavioral retention; (3) Beehiiv milestone referrals → viral lift; (4) Instagram Story-tag → the wedge action. **Highest-ROI platforms for SMB advocacy in 2026:** Instagram (local discovery) + TikTok (organic reach), with strict commercial-content/disclosure enforcement. **Platform-policy risks:** Meta API closure (verification gating), TikTok/IG incentivized-content rules, Reddit/Google/Yelp incentive bans — all of which become *features* when enforced by design.

## Part B — The direct competitive set (the real fight)

The real competitors are SMB word-of-mouth/UGC/loyalty/referral/ambassador SaaS — not the social platforms.

### Positioning map (two axes: what's rewarded × who it's for)
- **Reviews, local SMB:** Birdeye (~$299/mo), Podium ($399–599 real-spend), NiceJob ($75–125 flat) — own the local buyer, do **only reviews** (and incentivized reviews are now radioactive).
- **Purchases, local SMB:** Square Loyalty ($49), Toast, TapMango, Marsello — own redemption, reward **only purchases**.
- **Content/UGC, DTC mid-market:** Brandbassador ($500–3,000/mo), Roster ($599/mo), SocialLadder, SocialSnowball ($249+3%) — run the **exact loop** but for DTC ecommerce on annual contracts, 10–60× the price.
- **Shopify loyalty:** Smile.io ($15/$49/$199, 4,269 reviews, since 2014) — unassailable; **don't compete there.**
- **The empty quadrant:** *content/UGC × local SMB × self-serve × $29–79/mo* — **nobody is here.**

**Pricing verdict:** Social Perks sits at the *bottom* of the proven band — **pricing is not the barrier**; trust, proof, and integration are. The recommended $39/$99 v2 is still below the demonstrated ceiling.

### Whitespace & the defensibility thesis
- **Whitespace (empirically empty):** self-serve "reward local customers for social posts, redeem at the counter" at $29–79/mo.
- **Why it's stayed empty / what kills entrants:** two-sided cold-start (Belly/LevelUp/Perka graveyard), feature-absorption by incumbents (Square bundled loyalty 2025; Smile/Yotpo can add missions), and the Gatsby precedent (standalone reward-for-post became a *feature*, not a company). Fraud (coupon leaking, post-then-delete, fake engagement) is table stakes.
- **The moat Social Perks can actually build (ranked):**
 1. **Compliance-by-design** — structural FTC-safe flows + enforced disclosure. Every SMB faces $53K/violation; incumbents deliberately won't touch incentivized UGC. Truthful, defensible, and *needed*.
 2. **Real Meta-verified post checking** — a gated capability since Meta shut the Basic Display API (Dec 2024); "Meta approved" is a headline differentiator competitors market.
 3. **Local redemption UX + POS data** — the counter-redemption + transaction attribution incumbents-on-one-side lack.
 4. **Local cross-business network effect** — the Substack-Recommendations mechanic; the one moat that compounds and that no loyalty competitor has.
- **Distribution must be borrowed, not built:** winners ride rails (NiceJob via Jobber/Housecall/QuickBooks; Marsello via Lightspeed/Shopify POS; Birdeye via agency resellers). Social Perks' equivalent: **Square/Clover/Toast marketplaces + agency white-label**.
- **Timing:** influencer spend $32.55B (2026), 45.5% to nano/micro, 73% of brands prefer them; Later bought Mavely for $250M. The "customer-as-nano-creator" wave is cresting — **the whitespace won't stay empty**, so speed to the compliance + POS + network-effect moat matters.

> **Net competitive read:** Social Perks is not entering a crowded fight — it's entering an *empty quadrant adjacent to* crowded fights, with a regulatory tailwind that makes its exact lane (compliant UGC-for-perks) the one incumbents can't follow into without re-architecting. The win condition is to own that quadrant fast via borrowed distribution (Square + agencies) and lock it with compliance + verification + the local network effect before an incumbent bundles a watered-down version.
